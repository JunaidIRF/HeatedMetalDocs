# Spawnpoints

### SpawnPoints can be found under `Library` -> `Right Click` -> `Spawnpoint`

<figure><img src="../.gitbook/assets/image (19).png" alt=""><figcaption></figcaption></figure>

### If the current map does not have spawnpoints for the current game mode use the external menu to drag and place new spawnpoints.

<figure><img src="../.gitbook/assets/image (20).png" alt=""><figcaption></figcaption></figure>

### Most spawnpoints are named under ATK or DEF or hostage

### There are multiple spawnpoints for multiple gamemodes

* **PB** (Generic spawns for Hostage, Bomb and Secure)
* **Warmup** (Init, Respawn)
* **Kill Confirmed** spawnpoints are called **TDM** (Init, Respawn)
