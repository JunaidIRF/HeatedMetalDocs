# Installing Heated Metal

First of all make sure you have Shadow Legacy downloaded. You can find information about downloading any season on [this page](https://faq.r6downloads.com/r6-operation-throwback-help-page/how-to-download) and make sure you are in the [Operation Throwback discord server](https://discord.gg/JGA9WPF4K8) for updates and for help of any old season. You can follow the given steps to install `Heated Metal` after you have the game downloaded.

## Installing Heated Metal:

1. Download [HeliosLoader](https://discord.com/channels/1321476389815324733/1413624100727033926/1413638851918041232) from the [Heated Metal Discord](https://discord.gg/7mR9VxBxWd) releases channel. (`HeliosLoader.7z`)
2. (Recommended) Download latest release of [HeatedMetal Neon Dawn](https://github.com/DataCluster0/HeatedMetal/releases/latest) or [HeatedMetal Shadow Legacy (v0.2.3)](https://github.com/DataCluster0/HeatedMetal/releases/0.2.3) from github. (`HeatedMetal.7z`)
3. Extract `HeliosLoader.7z`  and replace the files in the game folder.
4. Extract `HeatedMetal.7z` and replace the files in the game folder.
5. Change your username by opening `HeliosLoader.json` with notepad and replacing `Mr_ConfigMan` with your username.
6. Run the game using `RainbowSix.exe`.
