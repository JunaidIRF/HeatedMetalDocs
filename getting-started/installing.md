# Installing Heated Metal

First of all make sure you have Shadow Legacy downloaded. You can find information about downloading Shadow Legacy on [this page](https://faq.r6downloads.com/r6-operation-throwback-help-page) and make sure you are in the [discord server](https://discord.gg/JGA9WPF4K8) for updates and for help. You can follow the given steps to install `Heated Metal` after you have the game downloaded.

## Installing Heated Metal:

1. Download [HeliosLoader](https://discord.com/channels/1092820800203141130/1335739761670754395) from their [discord](https://discord.gg/JGA9WPF4K8) releases channel. (`HeliosLoader.7z`)
2. Download latest release of [HeatedMetal](https://github.com/DataCluster0/HeatedMetal/releases/latest) from github. (`HeatedMetal.7z`)
3. Extract `HeliosLoader.7z`  and replace the files in the game folder.
4. Extract `HeatedMetal.7z` and replace the files in the game folder.
5. Change your username by opening `HeliosLoader.json` with notepad and replacing `Mr_ConfigMan` with your username.
6. Run the game using `RainbowSix.exe`.
